# College Event Feedback System

A modern web application for collecting, managing, and analyzing feedback from college events. Built for streamlined event evaluation and continuous improvement.[1]

## Features

- Event creation and management for organizers
- Real-time feedback submission from attendees
- Rating systems and comment collection
- Feedback analytics and reporting dashboard
- Responsive design for mobile and desktop use[2]

## Tech Stack

- **Frontend**: Next.js 14+, React, TypeScript
- **Styling**: Tailwind CSS, ShadCN UI components
- **Backend**: Supabase (PostgreSQL + Auth)
- **Deployment**: Vercel/Netlify
- **Other**: GitHub Actions for CI/CD


### For Organizers
- Create new events with details (name, date, description)
- Share feedback link with attendees
- View live analytics during events
- Export reports post-event

### For Attendees
- Submit ratings (1-5 stars) and comments
- Multiple choice questions for quick feedback
- Anonymous submission option[3]
